# Saqlain Qasim Portfolio

Static personal portfolio website built with HTML, CSS, and vanilla JavaScript.

## Pages

- `index.html` - homepage with hero, skills, selected work, PM profile preview, certifications, and CTA.
- `portfolio.html` - filterable CMS portfolio with demo case-study modals.
- `coordination.html` - project coordination and management profile.
- `resume.html` - combined resume preview and downloadable PDF.
- `contact.html` - Formspree-powered contact form and direct contact links.

## Deploy

Upload the folder contents to a GitHub Pages repository. No build step, framework, CMS, or server dependency is required.
